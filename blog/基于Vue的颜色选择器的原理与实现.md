## 渐变色相关操作

参考[less](https://github.com/less/less.js)的`mix`函数

```javascript
getGradientColor(color1, color2, weight) {
    const w1 = weight;
    const w2 = 1 - w1;
    const rgb = {
        r: Math.round(color1.r * w1 + color2.r * w2),
        g: Math.round(color1.g * w1 + color2.g * w2),
        b: Math.round(color1.b * w1 + color2.b * w2),
    };

    return rgb;
},
```
