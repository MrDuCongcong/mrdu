# package.json字段全解析

对于一个包来说，必须包含package.json文件。package.json文件存在的意义就是为了以便他人安装和管理包，其包含是描述包的元数据。

​    package.json文件的作用：

1. 列出该包所依赖的其他包。
2. 使用语义版本控制规则描述包的版本。
3. 使你的包具有重用性。更容易与他人共享。
