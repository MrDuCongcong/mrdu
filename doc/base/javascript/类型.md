#### 对象的属性

对象属性使用键值来标识，键值只能为字符串或Symbol值。

```javascript
const obj = {};
obj.null = 1;
obj.undefined = 2;
obj[3] = 3;
obj[true] = 4;
obj[false] = 5;

Object.keys(obj).forEach(item => {
    console.log("keys", item, typeof item);
});
```

```shell
> node hello.js
keys 3 string
keys null string
keys undefined string
keys true string
keys false string
```
